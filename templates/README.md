# templates

Forge contract, resolution policy, schema contract, and bundled-mode
integration templates.

| Template | Purpose |
| --- | --- |
| `forge_contract.yml` | Dataset registration and transformation configuration |
| `resolution_policy.yml` | Entity resolution rules and confidence thresholds |
| `schema_contract.yml` | Schema enforcement and versioned evolution |
| `bundled_mode.yml` | Optional DriftSentinel integration configuration (standalone by default) |
