# .claude/ — Claude Code Configuration Root

This directory contains all Claude Code plugin configuration for
the AetheriaForge repository.

## Contents

| Path | Purpose |
| --- | --- |
| `agents/` | 5 specialized subagent role definitions |
| `commands/` | 21 reusable command prompts (slash commands) |
| `hooks/` | Hook handlers and session logging |
| `settings.json` | Claude Code plugin configuration (checked in) |
| `settings.local.json` | Local overrides (not checked in) |
| `agent-memory/` | Agent runtime memory (not checked in) |
