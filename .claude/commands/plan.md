# plan

Create a structured ÆtheriaForge implementation or design plan.

## Variables

prompt: $ARGUMENTS

## Instructions

- Start with `CLAUDE.md`, `README.md`, and the canonical `specs/AF-*.md` suite.
- If the task needs a new plan file, create it under `specs/` using a concise,
  descriptive filename.
- Keep the plan aligned to the ÆtheriaForge taxonomy in
  `specs/AF-SDD-001_Architecture_Blueprint.md`.
- Include validation steps and explicit acceptance criteria.

## Report

Return only the path to the created or updated plan file.
